# Troubleshooting

## Common Issues

### HSM Connection Failed
- Check VPC security groups
- Verify HSM client configuration
- Check HSM is in ACTIVE state

### Authentication Failed
- Verify crypto user credentials
- Check user is activated
- Review HSM audit logs

### Key Operation Failed
- Check user has required permissions
- Verify key exists and is extractable
- Review operation parameters

### Cluster Degraded
- Check HSM health status
- Verify network connectivity
- Add replacement HSM if needed
